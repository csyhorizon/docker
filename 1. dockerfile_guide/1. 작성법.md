## 명령어
- 아래 12개만 익혀두면 대다수 커버 가능

| Command   | 기능                          | 비고              |
| ---       | ---                           | ---               |
| FROM      | 이미지 지정                   | Dockerfile 시작      |
| WORKDIR   | 작업 디렉토리 설정                | `cd`와 비슷 |
| COPY      | 복사                          | 소스 코드, 설정 파일 복사 시 사용 |
| ADD       | 복사 + 압축 해제 + URL 다운로드   | COPY보다 쩔지만 보안 주의 |
| RUN       | 쉘 명령 실행                  | 패키지 설치, 빌드 수행 (&& 동시 처리가 좋음) |
| CMD       | 시작 시 기본 명령             | `docker run` 시 인자 덮어쓰기 가능 |
| ENTRYPOINT| 시작 시 고정 명령             | 실행 파일 지정 시 사용 |
| ENV       | 환경 변수                     | 컨테이너 실행 중에도 환경변수 유지 |
| ARG       | 빌드 변수                     | 빌드 완료 후 이미지에 환경변수 안남음 |
| EXPOSE    | 문서용 포트                   | 문서용임 |
| USER      | 실행 계정 설정                | `root` 계정은 보안상 안쓰는 게 좋음 |
| VOLUME    | 마운트 지점                   | 컨테이너 삭제되도 데이터 유지 |
